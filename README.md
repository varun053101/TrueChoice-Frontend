# TrueChoice - Frontend

TrueChoice is a secure and transparent online voting platform for college clubs, student councils, and academic organizations.

## Project info

**URL**: https://truechoice.app

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/varun053101/TrueChoice-Frontend
   cd TrueChoice-Frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Deploy using Vercel, Netlify, or your preferred static site hosting provider.
